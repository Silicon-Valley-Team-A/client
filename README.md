# RECOMMADE FRONTEND
사진을 인식하여 사진 속의 사물 등을 추출하여 해당 사진의 환경이나 상황과 어울리는 노래를 추천해줍니다. 사용자는 추천 받은 노래들을 통해 플레이리스트를 생성할 수 있습니다.
